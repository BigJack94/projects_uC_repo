Automatic Watering Time Station 
	It is a development/updated of the project Automatic Watering Station for bigger watering systems.
Parameters:
-power supply 7-30V (12V is recommended)
-configuration by WiFi network
-RGB LED as indicator of state
-pushbutton to quick action 
-3 sensor input (12V input voltage, with galvanic isolation)
-2 power output to control relay
-RTC IC DS1307
-settings capabilities over Wi-Fi:
	-work mode (stop, work with 1 or/and 2 output, manual work)
	-network mode (station mode, access point, default access point, without wifi)
	-work times (4 start-stop times)
